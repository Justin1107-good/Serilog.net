 Serilog.net 应用程序的诊断日志记录库。它易于设置，具有干净的 API，并且可以在所有最新的 .NET 平台上运行。虽然 Serilog 即使在最简单的应用程序中也很有用，但在检测复杂、分布式和异步应用程序和系统时，它对结构化日志记录的支持却大放异彩。
